JadeAutoPost 隱私政策

生效日期：2026 年 6 月 21 日

JadeAutoPost（以下稱「本應用程式」）是供「岫玉雅集館」管理者使用的 Facebook 粉絲專頁自動發文工具。我們重視您的隱私，並依本政策說明資料的處理方式。

一、可能處理的資料

本應用程式可能透過 Meta 平台取得管理者帳號與粉絲專頁的識別資訊、授權範圍、粉絲專頁存取權杖，以及完成發文功能所必要的粉絲專頁內容。本應用程式不會要求或蒐集粉絲專頁訪客的密碼、付款資料或其他非功能所需的個人資料。

二、資料使用目的

上述資料僅用於驗證粉絲專頁管理權限、發布與管理粉絲專頁內容、維持自動發文功能，以及診斷發文失敗等技術問題。我們不會出售、出租或以廣告目的分享這些資料。

三、資料保存與安全

存取權杖僅保存於管理者控制的本機環境或 GitHub Secrets 等受保護的設定中，不會公開顯示於程式碼。資料只在提供功能所需期間內保存；授權遭撤銷、權杖失效或服務停止後，將停止使用相關資料。

四、第三方服務

本應用程式使用 Meta Graph API 與 GitHub Actions 提供授權、發文及排程功能。相關服務亦可能依其各自的隱私政策處理必要資料。

五、資料刪除與撤銷授權

您可在 Facebook 的「設定和隱私」中移除 JadeAutoPost 的商業整合或應用程式授權，以停止本應用程式存取資料。您也可寄信至 gigikuo0612@gmail.com 申請刪除相關資料；我們將在確認申請後 30 日內完成處理。

六、政策更新

本政策可能因功能或法規需求調整。更新內容會公布於本頁，並標示最新生效日期。

七、聯絡方式
